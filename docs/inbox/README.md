# inbox

Capture zone for untriaged ideas, expanded TODOs, and "we should look at this
someday" notes. Low friction in, triaged out.

- **Filename**: `YYYY-MM-DD-short-slug.md`.
- Write freely — a sentence or a page, whatever captures the idea.

## Triage

When an inbox item is picked up, decide its destination and move it out:

- Needs investigation before we can act → `research/`.
- Ready to specify → `product-specs/`.
- A pure technical decision → `decisions/` (ADR).
- Not worth doing → delete it (git history keeps the record).

The inbox should trend toward empty. It is a queue, not an archive.
